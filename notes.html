    A linear equation has the form:
    ðŸ”—
    \(\begin{aligned} a_1 x_1 + a_2 x_2 + \cdots + a_n x_n = b \end{aligned}\)
    ðŸ”—
    Here, \(a_1,\dotsc,a_n\) and \(b\) are the coefficients, \(x_1,\dotsc,x_n\) are the variables (unknowns), and \(n\) is the dimension (number of variables).
    ðŸ”—
    Examples:
    • \(2x_1+4x_2=4\) is a line in two dimensions.
      ðŸ”—
    • \(3x_1+2x_2+x_3=6\) is a plane in three dimensions.
      ðŸ”—
    ðŸ”—
    A linear system consists of multiple linear equations in the same variables. For example:
    ðŸ”—
    \(\begin{aligned} x_1 \amp\;+\; 1.5\,x_2 \;+\; \pi x_3 \;=\; 4 \\ 5x_1 \amp\;+\; \phantom{1.5\,x_2}\;+\; 7x_3 \;=\; 5 \end{aligned}\)
    ðŸ”—
    Definition 1.1.1. Solution to a Linear System.
    The set of all values of \(x_1, x_2, \ldots, x_n\) that satisfy every equation in the system.
    ðŸ”—
    ðŸ”—
    A system can have a unique solution, no solution, or infinitely many solutions.

    Definition 1.2.1. Definition: Echelon Form and RREF.
    A rectangular matrix is in echelon form if:
    ðŸ”—
    1. All zero rows (if any) are at the bottom.
      ðŸ”—
    2. The first nonzero entry (the leading entry) of a row is to the right of any leading entry in a row above it.
      ðŸ”—
    3. All entries below a leading entry are zero.
      ðŸ”—
    A matrix in echelon form is in row reduced echelon form (RREF) if additionally:
    ðŸ”—
    1. All leading entries equal \(1\text{.}\)
      ðŸ”—
    2. Each leading entry is the only nonzero entry in its column.
      ðŸ”—
    ðŸ”—
    Example 1.2.2. Example: A Matrix in Echelon Form.
    Let \(\blacksquare\) denote a nonzero number, and \(\ast\) denote any number.
    ðŸ”—
    \(\begin{bmatrix} 0 \amp \blacksquare \amp \ast \amp \ast \amp \ast \amp \ast \amp \ast \amp \ast \amp \ast \amp \ast \\ 0 \amp 0 \amp 0 \amp \blacksquare \amp \ast \amp \ast \amp \ast \amp \ast \amp \ast \amp \ast \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp \blacksquare \amp \ast \amp \ast \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp \blacksquare \amp \ast \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\)
    ðŸ”—

    Definition 1.2.3. Definition: Pivot Position and Pivot Column.
    A pivot position in a matrix \(A\) is a location in \(A\) that corresponds to a leading \(1\) in the RREF of \(A\text{.}\) A pivot column is a column of \(A\) that contains a pivot position.
    ðŸ”—
    ðŸ”—
    (a) Compute RREF and Identify Pivot Columns.
    Express the matrix in RREF and identify the pivot columns.
    ðŸ”—
    \(\left[ \begin{array}{rrrr} 0 \amp -3 \amp -6 \amp 4 \\ -1 \amp -2 \amp -1 \amp 3 \\ -2 \amp -3 \amp 0 \amp 3 \end{array} \right]\)
    ðŸ”—
    ðŸ”—
    Row Reduction Algorithm (Steps to RREF).
    The algorithm below produces a matrix in RREF.
    ðŸ”—
    1. Step 1a: Swap the first row with a lower one so the leftmost nonzero entry is in the first row.
      ðŸ”—
    2. Step 1b: Scale the first row so that its leading entry equals \(1\text{.}\)
      ðŸ”—
    3. Step 1c: Use row replacement so all entries below this \(1\) are \(0\text{.}\)
      ðŸ”—
    4. Step 2a: Swap the second row with a lower one so the leftmost nonzero entry below the first row is in the second row.
      ðŸ”—
    5. Continue: Repeat to obtain echelon form with leading entries equal to \(1\text{.}\)
      ðŸ”—
    6. Last step: Use row replacement so all entries above each leading entry are \(0\text{,}\) starting from the right.
      ðŸ”—
    ðŸ”—
    Example 1.2.4. Basic and Free Variables.
    Consider the augmented matrix \(\left[\,A\,\middle|\,\vec b\,\right]\text{:}\)
    ðŸ”—
    \(\left[ \begin{array}{ccccc|c} 1 \amp 3 \amp 0 \amp 7 \amp 0 \amp 4 \\ 0 \amp 0 \amp 1 \amp 4 \amp 0 \amp 5 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp 6 \end{array} \right]\)
    ðŸ”—
    The leading ones are in the first, third, and fifth columns. Thus, the pivot variables of the system \(A\,\vec x \,=\, \vec b\) are \(x_1\text{,}\) \(x_3\text{,}\) and \(x_5\text{;}\) the free variables are \(x_2\) and \(x_4\text{.}\) Any choice of the free variables leads to a solution of the system.
    ðŸ”—
    Note: The matrix \(A\) itself does not have â€œvariablesâ€; systems have variables.
    ðŸ”—
    ðŸ”—
    Theorem 1.2.5. Theorem: Existence and Uniqueness of Solutions.
    A linear system is consistent iff the last column of the augmented matrix does not have a pivot. Equivalently, the RREF of the augmented matrix does not contain a row of the form \(\left[\,0,\,0,\,0,\,\dots,\,0 \,\middle|\, 1\,\right]\text{.}\)
    ðŸ”—
    If a linear system is consistent, then:
    ðŸ”—
    1. It has a unique solution iff there are no free variables.
      ðŸ”—
    2. Otherwise, it has infinitely many solutions parameterized by the free variables.
      ðŸ”—
