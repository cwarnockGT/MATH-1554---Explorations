    Section Exploration 8, due March 22

    Handout Exploration 8, due March 22
    

    Make sure to familiarize yourself with the Exploration Instructions found on the home page.
    🔗
    In this exploration, we examine how the linear algebra tools developed in \(\mathbb{R}^n\text{,}\) including linear transformations, eigenvalues, and eigenvectors, extend naturally to more abstract vector spaces such as spaces of polynomials.
    🔗

    Problem 1.

    Let \(\mathbb{P}_n\) denote the \((n+1)\)-dimensional vector space of polynomials of degree at most \(n\text{;}\) that is, let
    \begin{equation*} \mathbb{P}_n = \left\{a_nx^n + a_{n-1}x^{n-1} + \cdots a_1x + a_0 \ : \ a_i \in\mathbb{R}\right\}\text{.} \end{equation*}
    It follows that \(\mathbb{P}_1\) is the 2-dimensional vector space of all linear (and constant) polynomials. Consider the linear transformation \(T:\mathbb{P}_1\to\mathbb{P}_1\) defined by
    \begin{equation*} T(p(x)) = p(0)x + p(2)\text{.} \end{equation*}
    So \(T\) takes a linear polynomial as an input and produces a linear polynomial as an output. For example, if \(p(x)=-5x+6\text{,}\) then \(p(0)=6\) and \(p(2)=-4\text{,}\) so
    \begin{equation*} T(p(x)) = T(-5x + 6) = p(0)x + p(2) = 6x-4\text{.} \end{equation*}

    Problem 2.

    Just as in \(\mathbb{R}^n\text{,}\) we may choose a basis for \(\mathbb{P}_1\text{.}\) Consider the following basis for \(\mathbb{P}_1\text{:}\)
    \begin{equation*} \mathcal{B}=\{-x+1,\;2x+1\}. \end{equation*}
    We define the coordinate vector of \(p(x) \in \mathbb{P}_1\) with respect to the basis \(\mathcal{B}\) in the same way we do for \(\mathbb{R}^n\text{.}\) So, if
    \begin{equation*} p(x)=a(-x+1)+b(2x+1) \end{equation*}
    with \(a,b\in\mathbb{R}\text{,}\) its coordinate vector is:
    \begin{equation*} \left[p(x)\right]_{\mathcal{B}}=\begin{bmatrix} a \\ b\end{bmatrix}. \end{equation*}
    This relationship is how we can use our Linear Algebra knowledge about \(\mathbb{R}^n\) to determine results about the more abstract vector spaces.

    Problem 3.

    The \(\mathcal{B}\)-matrix of \(T\) is given by
    \begin{equation*} T_\mathcal{B} = \begin{bmatrix} \left[T(-x+1)\right]_{\mathcal{B}} \amp \left[T(2x+1)\right]_{\mathcal{B}} \end{bmatrix}\text{.} \end{equation*}
    Determine \(T_\mathcal{B}\) and find its eigenvalues. Hint: \(T_\mathcal{B}\) should be a \(2 \times 2\) upper triangular matrix.
