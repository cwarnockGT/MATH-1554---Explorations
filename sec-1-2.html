    Definition 1.2.1. Echelon Form and Row Row Reduced Echelon Form.

    A rectangular matrix is in echelon form (REF) if:
    ðŸ”—
    1. All zero rows (if any) are at the bottom.
      ðŸ”—
    2. The first nonzero entry (or leading entry) of a row is to the right of any leading entry in the row above it.
      ðŸ”—
    3. All entries below a leading entry are zero.
      ðŸ”—
    A matrix in echelon form is in row reduced echelon form (RREF) if additionally:
    ðŸ”—
    1. All leading entries equal \(1\text{.}\)
      ðŸ”—
    2. Each leading entry is the only nonzero entry in its column.
      ðŸ”—
    ðŸ”—

    Definition 1.2.4. Pivot Position and Pivot Column.

    A pivot position in a matrix \(A\) is a location in \(A\) that corresponds to a leading \(1\) in the RREF of \(A\text{.}\) A pivot column is a column of \(A\) that contains a pivot position.
    ðŸ”—
    ðŸ”—

    Row Reduction Algorithm (Steps to RREF).

    The algorithm below produces a matrix in RREF.
    ðŸ”—
    • Step 1a: Swap the first row with a lower one so the leftmost nonzero entry is in the first row.
      ðŸ”—
    • Step 1b: Scale the first row so that its leading entry equals \(1\text{.}\)
      ðŸ”—
    • Step 1c: Use row replacement so all entries above and below this \(1\) are \(0\text{.}\)
      ðŸ”—
    • Step 2a: Swap the second row with a lower one so the leftmost nonzero entry below the first row is in the second row.
      ðŸ”—
    • Step 2b: Scale the second row so that its leading entry equals \(1\text{.}\)
      ðŸ”—
    • Step 2c: Use row replacement so all entries above and below this \(1\) are \(0\text{.}\)
      ðŸ”—
    • Continue: Repeat these steps for each subsequent row until the matrix is in RREF.
      ðŸ”—
    ðŸ”—

    Definition 1.2.7. Pivot Variables and Free Variables.

    Variables in a system that correspond to pivot column in the augmented matrix are called pivot variables. The other variables in the system are called free variables. This is because, if the system is consistent, then any choice of the free variables will lead to a valid solution.

    Theorem 1.2.9. Existence and Uniqueness of Solutions.

    A linear system is consistent if and only if the last column of the augmented matrix does not have a pivot. Equivalently, the RREF of the augmented matrix does not contain a row of the form
    \begin{equation*} \begin{bmatrix} 0 \amp 0 \amp 0 \amp \cdots \amp 0 \amp \vert \amp 1\end{bmatrix}\text{.} \end{equation*}
    ðŸ”—
    If a linear system is consistent, then:
    ðŸ”—
    1. It has a unique solution if and only if (iff) there are \(\underline{\phantom{free variable free variables}}\text{.}\)
      ðŸ”—
    2. Otherwise, it has \(\underline{\phantom{infinitely infinitely}}\) many solutions parameterized by the free variables.
